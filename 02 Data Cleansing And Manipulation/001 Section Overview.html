This section focuses on Data Cleansing & Manipulation

The Data cleansing part of this section will focus on:

  • Functions that can deal with null values, specifically NVL() and Coalesce()

  • Functions that can clean values, specifically Trim(), LPAD() and RPAD()

The Data Manipulation part of this course will show you how to PIVOT and UNPIVOT your data, this is very useful if you want to have a pivoted view of your table or if you want to unpivot a pivoted table view.
